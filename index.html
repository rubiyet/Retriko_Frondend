Utopic is a photo viewer and storage service application for modern Windows Operating System. It was developed with C# in front and PHP in the backend featuring natural language search powered by Microsoft Azure cognitive service.

Theia is an all-in-one Optical Mark Reader solution. Speed and accuracy makes it unique from other applications of its kind. It uses OpenCV library and Microsoft .NET framework to deliver highest efficiency along with most friendly user experience.
